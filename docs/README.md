1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Set up environment variables (see [Resend Setup](./RESEND_SETUP.md)):
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your Resend API key
   ```

3. Run development server:
   ```bash
   pnpm dev
   ```

4. Build for production:
   ```bash
   pnpm build
   ```
